# Reports

Use this directory for exploratory output generated during a specific analysis run.

Guidelines:

- Create a dated subdirectory such as `reports/run-2026-04-01/`.
- Keep rough findings, inventories, unresolved questions, and intermediate notes here.
- Promote only stable, reviewed conclusions into `docs/`.
- Prefer seeding recurring report files from:
  - `templates/run-inventory-template.md`
  - `templates/open-questions-template.md`
