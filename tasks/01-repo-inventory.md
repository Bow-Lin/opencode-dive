# Task 01: Repository Inventory

## Goal

Build an initial map of the pinned Opencode source tree and identify likely entrypoints, build metadata, and major subsystems.

## Input Scope

- `workspace/source/`
- top-level package/build/config files
- source root directories

## Must Read

- `OPENCODE_VERSION.md`
- `docs/{en,zh}/analysis-rules.md`
- `scripts/tree.sh`
- `scripts/generate-file-index.py`

## Expected Output

- run-scoped notes in `reports/run-YYYY-MM-DD/inventory.md`
- updates to `docs/{en,zh}/repo-map.md`

## Completion Standard

- top-level directories are mapped,
- build/runtime metadata files are listed,
- likely entrypoints are identified,
- and unknown areas are recorded explicitly.
